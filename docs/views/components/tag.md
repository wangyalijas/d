
# Tag 标签

## 基础用法
 <m-tag />

## 可移除标签
<m-tag-remove />

## 尺寸选择
<m-tag-size />

## Attributes

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| type | 类型 | String | primary / success / info / warning / error | primary |
| size | 大小 | String | medium / small / mini | normal |
| closable | 是否可关闭 | Boolean | - | false |

## Events

| 参数      | 说明          | 回调参数 |
|---------- |-------------- |---------- |
| close | 关闭tag时触发的事件 | - |
