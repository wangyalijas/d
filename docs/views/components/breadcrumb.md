# Breadcrumb 面包屑 (✓)

## 基础用法

<m-breadcrumb />

## 图标分隔符

<m-breadcrumb-icon />

## Breadcrumb Attributes

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| separator | 分隔符号 | String | - | / |
| separator-icon-class | 分割图标class | String | - | - |

## BreadcrumbItem Attributes

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| jump-to | 跳转的路由信息, 同vue-router的to | String/Object | - | - |
